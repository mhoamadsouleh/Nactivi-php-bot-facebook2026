# WhatsApp Bot for Djezzy

بوت فيسبوك لتسجيل وعروض جيزي

## المميزات
- تسجيل أرقام جيزي
- تفعيل العروض تلقائياً
- نظام دعوة الأصدقاء
- حفظ البيانات في SQLite

## التقنيات المستخدمة
- PHP 8.1
- SQLite
- Facebook Graph API
- Docker

## النشر على Render
1. انسخ هذا المشروع إلى GitHub
2. سجل في Render.com
3. أنشئ خدمة ويب جديدة
4. اختر هذا المستودع
5. استخدم إعدادات Docker
